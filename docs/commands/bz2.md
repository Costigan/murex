# _murex_ Shell Guide

## Command Reference: `!bz2` (optional)

> Decompress a bz2 file

### Description

`!bz2` is an optional builtin for decompressing a bz2 stream from STDIN.

### Usage

    <stdin> -> !bz2 -> <stdout>

### Detail

Currently there is no support for compressing a stream using bz2.

### Synonyms

* `!bz2`
