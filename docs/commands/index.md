# _murex_ Language Guide

## Command Reference: `[` (index)

> Outputs an element from an array, map or table

### Description

Outputs an element from an array, map or table.

### Usage

    <stdin> -> [ element ] -> <stdout>
    $variable[ element ] -> <stdout>
    
    <stdin> -> ![ element ] -> <stdout>

### Synonyms

* `[`
* `![`
